Ejemplo de Control de Flujo de Texto

White-space: normal. Este es un ejemplo de texto largo que se ajusta automáticamente al ancho del contenedor.

White-space: nowrap. Este texto no permite saltos de línea y se mantiene en una sola línea horizontal, incluso si se desborda.

Text-overflow: ellipsis. Este texto se truncará con puntos suspensivos si no cabe en el contenedor.

Overflow-wrap: break-word. Una palabraextraordinariamentelargaquedesbordaría se ajustará para no romper el diseño.

Line-clamp: este texto estará limitado a tres líneas. Si excede ese límite, se mostrará una elipsis al final.
