Ejemplo de Desbordamiento

Overflow: visible - Este contenido desborda del contenedor y es completamente visible.

Overflow: hidden - Este contenido desborda del contenedor, pero la parte que excede no se muestra.

Overflow: scroll - Este contenido desborda del contenedor y puedes desplazarte para verlo.

Overflow: auto - Este contenido desborda del contenedor y aparecen barras de desplazamiento solo si es necesario.
